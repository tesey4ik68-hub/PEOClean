# Установщик программы "Учёт дворников"

## Описание
Этот установщик создан с помощью INNO Setup 6 и позволяет установить программу "Учёт дворников" на любой компьютер с Windows без необходимости установки дополнительных компонентов.

## Особенности установщика
- **Самодостаточный**: Включает все необходимые компоненты .NET 9.0
- **Установка для текущего пользователя**: Не требует прав администратора
- **Русский интерфейс**: Установщик на русском языке
- **Создание ярлыков**: Автоматически создает ярлыки в меню "Пуск" и на рабочем столе (опционально)

## Системные требования
- Windows 10 или новее
- Минимум 2 ГБ оперативной памяти
- Минимум 500 МБ свободного места на диске

## Установка
1. Запустите файл `installer_output/PEOclean_Installer.exe`
2. Следуйте инструкциям установщика
3. Выберите папку для установки (по умолчанию: `C:\Users\[Имя пользователя]\AppData\Roaming\Учёт дворников`)
4. При необходимости отметьте создание ярлыка на рабочем столе
5. Нажмите "Установить"

## После установки
- Программа будет доступна в меню "Пуск" → "Учёт дворников"
- Ярлык на рабочем столе (если был выбран при установке)
- Программа готова к работе без дополнительных настроек

## Удаление программы
- Используйте стандартную функцию "Удаление программ" в Windows
- Или запустите установщик повторно и выберите "Удалить"

## Технические детали
- **Размер установщика**: ~52.1 МБ (с иконкой и обновленным окном "О программе")
- **Размер установленной программы**: ~500 МБ
- **Архитектура**: x64
- **.NET версия**: 9.0 (включена в установщик)
- **База данных**: SQLite (создается автоматически при первом запуске)

## Сборка установщика в VS Code
Для быстрой сборки установщика в VS Code:
1. Нажмите `F1` (или `Ctrl+Shift+P`)
2. Выберите "Tasks: Run Task"
3. Выберите "build-installer"

Установщик будет автоматически опубликован в папку `installer_output/`.
